# Enzyme Scraper

This repository contains a pipeline for downloading papers from [NCBI](https://www.ncbi.nlm.nih.gov) and GNU zip ```.gz``` file extractor. Additionally, an NLP pipeline for extracting enzyme names is available.

## Requirements

```
ftputil==4.0.0
textacy==0.10.1
spacy==2.3.2
pandas==0.24.2
```
